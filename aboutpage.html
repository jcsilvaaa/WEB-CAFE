About TasteCheck

Discover the core technologies behind TasteCheck.

Technologies Used

Express.js

A minimal and flexible Node.js framework for building web applications and APIs.

Mongoose

An ODM library for MongoDB, providing schema-based models and easy data validation.

CORS

Middleware for enabling Cross-Origin Resource Sharing, allowing secure API requests from different origins.

Multer

Middleware for handling multipart form-data, primarily used for file and image uploads.

Path

Built-in Node.js module for handling and transforming file and directory paths.

Bcrypt

A library for hashing passwords, enhancing security with encryption.

Why These Libraries?

We use these core libraries to build and enhance TasteCheck's functionality. Express.js powers our server, Mongoose connects us to MongoDB, and CORS handles secure API communication. Multer makes file uploads seamless, Path simplifies file management, and Bcrypt encrypts user passwords for added security.
